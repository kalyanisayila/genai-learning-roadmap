# Week 1 - Day 3: Tokens, Prompts, Context Window, Hallucinations & Prompt Engineering

## What are Tokens?

Tokens are small pieces of text that an AI reads and processes. A token can be a word, part of a word, number, or punctuation mark.

### Example
Sentence: "I love Python!"

Tokens may be: I | love | Python | !

### Easy Example
Tokens are like LEGO blocks. AI uses these small blocks to understand and create sentences.

---

## What is a Prompt?

A prompt is the instruction or question given to an AI tool.

### Example
"Write a Python program to add two numbers."

A clear and specific prompt gives a better answer.

---

## What is a Context Window?

A context window is the amount of text an AI can consider at one time. It includes the current prompt and earlier messages in a conversation.

### Example
If I first say, "I am learning Python," and later ask, "Give me a study plan," the AI can use the earlier information if it is inside the context window.

### Easy Example
A context window is like space on a whiteboard. When it becomes full, older information may not fit.

---

## What are AI Hallucinations?

AI hallucination happens when an AI gives false or made-up information that sounds believable.

### Example
An AI may give a fake research-paper title or an incorrect company CEO name.

### How to Avoid It
- Verify important facts using reliable sources.
- Ask for sources.
- Do not copy important information without checking.

---

## Prompt Engineering Basics

Prompt engineering means writing clear instructions so AI gives a useful answer.

### Formula
Role + Task + Context + Examples + Output Format

### Example Prompt

Role: Act as a Python teacher.

Task: Explain Python lists.

Context: I am a beginner preparing for fresher interviews.

Examples: Give definition, syntax, and a small program.

Output Format: Use simple English, headings, and bullet points.

---

## Key Takeaways

- Tokens are small pieces of text processed by AI.
- A prompt is an instruction given to AI.
- A context window is the text AI can consider at one time.
- Hallucinations are incorrect or made-up AI answers.
- Prompt engineering helps us write better instructions for AI.
