# Week 3 - Day 15: Getting a Google AI Studio API Key

## Objective
Learn how to create an API key for accessing Google's Gemini AI models.

## Steps
1. Open Google AI Studio.
2. Sign in with your Google account.
3. Click "Get API Key".
4. Create a new API key.
5. Copy and save the API key securely.

## Why Do We Need an API Key?
An API key allows Python applications to securely communicate with Gemini AI models.

## What I Learned
- How to access Google AI Studio.
- How to generate an API key.
- Why API keys should be kept private.

## Conclusion
Today I generated my first Google AI Studio API key, which will be used to build AI applications with Python.
