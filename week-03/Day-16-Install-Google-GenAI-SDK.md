# Week 3 - Day 16: Install Google GenAI SDK

## Objective
Learn how to install the Google GenAI SDK to use Gemini models in Python.

## Install the SDK

Open your terminal and run:

bash
pip install google-genai


## Verify the Installation

python
import google.genai

print("Google GenAI SDK installed successfully!")

## Why Do We Need the SDK?

The Google GenAI SDK allows Python applications to communicate with Gemini AI models using your API key.

- How to import the SDK in Python.
- Why the SDK is needed for AI applications.
