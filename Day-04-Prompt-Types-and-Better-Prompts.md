# Week 1 - Day 4: Prompt Types and Better Prompt Writing

## What are Prompt Types?

Prompt types are different ways of giving instructions to an AI tool.

## 1. Simple Prompt

A simple prompt gives a direct instruction.

### Example
"Explain Python lists."

---

## 2. Role-Based Prompt

A role-based prompt tells AI who it should act as.

### Example
"Act as a Python teacher and explain Python lists for a beginner."

---

## 3. Context-Based Prompt

A context-based prompt gives background information.

### Example
"I am a CSE student preparing for a fresher interview. Explain Python lists in simple English."

---

## 4. Step-by-Step Prompt

This prompt asks AI to explain a topic in steps.

### Example
"Explain how to create a Python function step by step with an example."

---

## 5. Output Format Prompt

This prompt tells AI how to present the answer.

### Example
"Explain Python lists using headings, 5 bullet points, and one program."

---

## Bad Prompt vs Better Prompt

### Bad Prompt
"Explain Python."

### Better Prompt
"Act as a Python teacher. Explain Python lists for a beginner preparing for interviews. Use simple English, headings, 3 examples, and 5 interview questions."

## Key Takeaways

- A prompt is an instruction given to AI.
- Clear prompts give better answers.
- Role, context, and output format improve prompts.
- Specific prompts help AI understand exactly what we need.
