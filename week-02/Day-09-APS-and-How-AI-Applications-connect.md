# Week 2 - Day 9: APIs and How AI Applications Connect

## What is an API?

API stands for **Application Programming Interface**.

An API is a bridge that allows two software applications to communicate and exchange data.

## Real-Life Example

Think of a restaurant.

- You are the customer.
- The waiter is the API.
- The kitchen is the AI model.

### Process

1. You tell the waiter what food you want.
2. The waiter takes your order to the kitchen.
3. The kitchen prepares the food.
4. The waiter brings the food back to you.

The waiter acts as the API, connecting you and the kitchen

## How AI Applications Use APIs

Example: A student asks ChatGPT:

> "Explain Python dictionaries."

The process is:

User
↓
Website or Mobile App
↓
API Request
↓
AI Model (LLM)
↓
API Response
↓
Website displays the answer

## Why APIs are Important

- Connect applications together.
- Send user requests to AI models.
- Receive AI-generated responses.
- Save development time.
- Enable integration with websites and mobile apps.


## Real-Life Examples

- ChatGPT API helps developers add AI features to websites.
- Weather apps use APIs to fetch live weather information.
- Payment apps use APIs to process online payments.
- Google Maps uses APIs to display maps and navigation.


## Key Takeaways

- API stands for Application Programming Interface.
- APIs allow different applications to communicate.
- AI applications use APIs to send prompts and receive responses.
- APIs are widely used in websites, mobile apps, and cloud services.
