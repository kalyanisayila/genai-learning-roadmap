## Explanation

API stands for **Application Programming Interface**.

An API acts as a bridge between two applications. It allows one application to request data or services from another application and receive a response.

### Example

Imagine ordering food in a restaurant.

- You = User
- Waiter = API
- Kitchen = Server

You give your order to the waiter. The waiter takes it to the kitchen and brings the food back to you.

Similarly, an API sends your request to a server and returns the response.

## Common HTTP Methods

- GET – Retrieve data
- POST – Send new data
- PUT – Update existing data
- DELETE – Remove data

## Why APIs Are Important

- Connect applications
- Exchange data quickly
- Used in websites and mobile apps
- Used by AI models like ChatGPT and Gemini

## Key Takeaways

- API is a communication bridge between applications.
- APIs work using requests and responses.
- HTTP methods include GET, POST, PUT, and DELETE.
- APIs are essential for building AI-powered applications.
