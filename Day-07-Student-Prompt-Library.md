# Week 1 - Day 7: Student Prompt Library

## Mini Project

This prompt library contains useful AI prompts for students. These prompts can help with learning, coding, interview preparation, resumes, and emails.

## 1. Learning Prompt

text
Act as a Python teacher. Explain Python functions for a beginner using simple English, syntax, and 3 examples.

## 2. Coding Prompt

text
Write a Python program to check whether a number is prime. Explain the code line by line.


## 3. Interview Preparation Prompt

text
Act as a software engineer interviewer. Ask me 10 Python interview questions for a fresher. Give feedback after each answer.

## 4. Resume Improvement Prompt

text
Act as an ATS resume reviewer. Improve my project description using action verbs and Python-related keywords. Keep it within 3 bullet points.


## 5. Email Writing Prompt

text
Write a professional one-day leave email to my college faculty because of fever. Keep it short and polite.

## 6. Notes Summary Prompt

text
Summarize the following notes in simple English using headings and 5 bullet points.

## 7. Study Plan Prompt

text
Create a 7-day Python study plan for a beginner preparing for fresher software-engineer interviews. Include daily topics and one coding practice question per day.


## Prompt Formula

text
Role + Task + Context + Output Format

## Example of a Strong Prompt

text
Role: Act as a Python teacher.

Task: Explain Python dictionaries.

Context: I am a beginner preparing for fresher interviews.

Output Format: Give a definition, syntax, 3 examples, and 5 interview questions.


## Learning Outcome

Through this mini project, I learned how to write clear prompts using role, task, context, and output format.
