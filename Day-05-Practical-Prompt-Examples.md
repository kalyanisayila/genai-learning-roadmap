# Week 1 - Day 5: Practical Prompt Examples

## Introduction

A good prompt is clear, specific, and tells the AI what output we need.

## 1. Prompt for Learning

Prompt:
"Act as a Python teacher. Explain Python functions for a beginner using simple English, syntax, and 3 examples."

## 2. Prompt for Coding

Prompt:
"Write a Python program to check whether a number is prime. Explain the code line by line."

## 3. Prompt for Interview Preparation

Prompt:
"Act as a software engineer interviewer. Ask me 10 Python interview questions for a fresher. Give answers after each question."

## 4. Prompt for Resume Improvement

Prompt:
"Act as an ATS resume reviewer. Improve this project description using action verbs and Python-related keywords. Keep it under 3 bullet points."

## 5. Prompt for Email Writing

Prompt:
"Write a professional one-day leave email to my college faculty because of fever. Keep it short and polite."

## 6. Prompt for Summarizing Notes

Prompt:
"Summarize the following notes in simple English using headings and 5 bullet points."

## Better Prompt Formula

Role + Task + Context + Output Format

## Example

Role: Act as a Python teacher.

Task: Explain Python dictionaries.

Context: I am a beginner preparing for fresher interviews.

Output Format: Give a definition, syntax, 3 examples, and 5 interview questions.

## Key Takeaways

- Specific prompts give better results.
- Tell AI the role, task, context, and output format.
- AI can help with learning, coding, resumes, emails, and interview preparation.
