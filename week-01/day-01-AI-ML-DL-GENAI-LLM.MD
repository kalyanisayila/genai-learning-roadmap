# Week 1 - Day 1: AI, ML, DL, GenAI & LLM

What is AI?
Artificial Intelligence (AI) enables machines to perform tasks that normally require human intelligence.

Example: Google Maps suggests the fastest route based on traffic.

What is Machine Learning (ML)?
Machine Learning is a subset of AI where computers learn patterns from data.

Example: YouTube recommends videos based on your watch history.

What is Deep Learning (DL)?
Deep Learning is a subset of Machine Learning that uses neural networks.

Example: Face Unlock on smartphones.

What is Generative AI?
Generative AI creates new content like text, images, code, and audio.

Example: ChatGPT writes emails, code, and answers questions.

What is an LLM?
LLM stands for Large Language Model.

An LLM is trained on massive amounts of text to understand and generate human-like language.

Example: ChatGPT uses an LLM to answer your questions.

Key Takeaways
- AI → Makes machines intelligent.
- ML → Learns from data.
- DL → Uses neural networks.
- GenAI → Creates new content.
- LLM → Understands and generates human language.
