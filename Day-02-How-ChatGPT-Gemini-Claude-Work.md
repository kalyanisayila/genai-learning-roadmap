# Week 1 - Day 2: How ChatGPT, Gemini, and Claude Work

## What are ChatGPT, Gemini, and Claude?

ChatGPT, Gemini, and Claude are AI assistants powered by Large Language Models (LLMs). They understand user prompts and generate human-like responses.

## How do they work?

### Step 1: User enters a prompt
Example:
"Write a leave email."

### Step 2: AI understands the prompt
The AI identifies what the user is asking.

### Step 3: LLM processes the request
The Large Language Model predicts the best response based on patterns learned during training.

### Step 4: AI generates the response
The AI creates and displays a complete answer.

## Real-Life Example

Input:
"Write Python code to add two numbers."

Output:
The AI generates the Python code within seconds.

## Key Points

- Powered by Large Language Models (LLMs)
- Understand user prompts
- Generate human-like responses
- Help with coding, writing, learning, and problem-solving
